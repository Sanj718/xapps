Features

Custom Widget Redirects

Overlay popup with redirect buttons. Customize colors, fonts, and layout. Control display frequency, geo targeting by country or continent, and advanced JS-based display rules.

Custom Auto Redirects

Automatic URL redirects with no popup shown. Configure custom destinations, redirect frequency, geo targeting, UTM parameters, and Google Analytics event tracking.

Markets Widget Redirects

Overlay popup powered by Shopify Markets. Syncs markets data automatically. Full appearance customization, geo targeting, and advanced display rules for widgets and items.

Markets Auto Redirects

Automatic URL redirects powered by Shopify Markets. Syncs markets data, geo-based targeting with country skip options, UTM parameters, and Google Analytics event tracking.
